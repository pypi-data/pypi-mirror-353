This module allows you to configure and display a specific phone number
for each Point of Sale (POS) on the printed sales receipt, replacing the
company's default phone number.

**Table of contents**

.. contents::
   :local:

Configuration
=============

Set up phone number of each POS and you will see that in every ticket on
the header

Usage
=====

1. Go to Configuration > Settings.
2. In the PoS Interface section, set up the phone number of the POS.
3. Once configured, enter the POS and make a sale. At the top of the
   printed ticket, you will see the phone number of the POS, not the
   companyâ€™s general phone number.
