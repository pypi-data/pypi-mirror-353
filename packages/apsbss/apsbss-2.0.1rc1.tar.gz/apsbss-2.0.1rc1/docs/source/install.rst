.. _install:

Install
=======

Follow these steps to install the ``apsbss`` package and executables.

.. code-block:: bash

    conda create -n apsbss python apsu::aps-dm-api pyepics
    conda activate apsbss
    pip install git+https://github.com/BCDA-APS/apsbss
